THE 6 AGENTS — WHAT EACH DOES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CROP DOCTOR → detects crop disease from photo/text → gives treatment
   - Input: WhatsApp image or text description
   - Model: PyTorch ResNet50 on PlantVillage dataset (optional) + watsonx.ai
   - Output: DISEASE / CAUSE / REMEDY / WARNING format
   - Saves to: disease_reports table + community_alerts aggregation

2. MANDI ORACLE → real-time crop price + sell/hold advice
   - Input: farmer asks about price of any crop
   - Data: Agmarknet API (data.gov.in)
   - Output: price in Rs/quintal + recommendation

3. VAYU GUIDE → weather-based farming advice
   - Input: question about irrigation, spraying, sowing
   - Data: OpenWeatherMap 3-day forecast
   - Output: specific action + timing

4. SCHEME ADVISOR → government scheme discovery
   - Input: farmer profile (state, crop, land size, income)
   - Data: 48 schemes in govt_schemes table
   - Output: top 3 matching schemes with apply URL

5. FARMER MEMORY → personalized context from past interactions
   - Reads: past diseases, last mandi query from DB
   - Injects into every watsonx prompt automatically
   - Makes bot remember "you had blight last season"

6. COMMUNITY INTEL → crowdsourced disease spread alerts
   - Reads: disease_reports aggregated by district
   - Uses: Neo4j graph for spread pattern detection
   - Output: "12 farmers in your area reported X"
